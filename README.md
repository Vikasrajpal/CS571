# Python Tkinter library based GUI program application used to manage and execute multiple operations related to order details on your system. User can perform operations like
```
"Customer Name"
"Contact Number"
"Bill NO."
```
```
Fast Food items
Grocery items
Taxes
Total Calculated Bill
Clear Field
Exit
```
